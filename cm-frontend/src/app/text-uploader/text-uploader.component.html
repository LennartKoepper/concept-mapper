Concept Map from Unstructured Text

Create a concept map from any unstructered text (e.g. an scientific abstract).

Shorter and more concise texts will usually yield better results. Choosing an appropriate context preset may also help. Depending on the selected model (and its context size), long texts may result in server-side processing errors, preventing their conversion into a concept map. A method to mitigate such errors is currently under development. However, since there seems to be no direct correlation between the context size and this behavior, it has not yet been possible to determine a specific length up to which texts can be processed without issues.
